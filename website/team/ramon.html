2 The Mandelbrot Set


2.2 The Mandelbrot Set

The Mandelbrot set [13] is related to the behaviour of following complex one dimensional map over infinite iterations: zk+1 = zk2 + c where z,c in C and z0 = 0. We say that the Mandelbrot Set is the set of points c in C that when plugged in Eq. (1), converge as k goes to infinity.

It is also important to remark what colours mean in this representation. The main light blue area represents the actual points that belong to the set, whereas the gradient of colours from blue to black is given by the number of iterations necessary to reach the escape condition.[14]

Lemma 2.2.1. The escape condition in the Mandelbrot Set is reached when at some point in the iterations the value of |zk|> 2 for some k. Proof. We intend to prove that |z|> 2 ⇒ |z2 + c|> |z| (i.e. diverges). First suppose that |z|> 2 which, invoking the function definition can be rewritten as |z2 + c|> 2. From the triangle inequality[15] we also know that |z2 + c|+|c|>= |z2|. Therefore:

|z2 + c| >= |z2|-|c| By the triangle inequality. |z2 + c| > 2|z|-|c| By the assumption that |z|> 2. (*) Now we need to show that |z|≥ |c| which can be obtained by considering the first iterations: |z1| = |c| |z2| = |c2 + c| = |c||c + 1|> |c| . |zn| > |c|

At this point it can easily be deduced that 2|z|-|c| >= |c| from the last result because |z| >= |c| ⇒ |z|-|c|>= 0 ⇒ 2|z|-|c| >= |z| >= |c|. This statement combined with (*) leads us to |z2 + c| > |z|.[16][17]

Saying that the Mandelbrot set is restricted to |zk| <= 2 is equivalent to saying that all the elements are contained in the disk of radius 2 around the origin, which implies it's bounded. Furthermore, from the lemma just proven, it is obvious that all the limit points[23][24] lie in the set. So we can conclude that the Mandelbrot set is a compact set. Finally, we'll demonstrate the relationship between the sequence of real points ck = ak + 0i in the Mandelbrot set and the period-doubling cascade ak of the logistic map.

Definition 2.2.4. Logistic map: Recurrence relation of degree 2 of the form xn+1 = rxn(1 - xn).[25] The relationship between logistic maps and the Mandelbrot set follows from a redefinition of the variables. We need to allow x, r in C and perform the following variable changes:
zn = r(xn - 1)/2
c = (1-(r-1)2)/4
Now, using simple arithmetic we can rewrite the equation:
xn+1 = rxn(1 - xn)
xn+1 = r(1/4 - (xn - 1/2)^2)^2
...
zn+1 = zn2 + c

At this point, we can intuitively deduce that both are representations of the same phenomenon. Let's consider the following graph, which we'll use to deepen our understanding about this relationship.

4.3 The algorithm

Based on [43] and [44], consider two independent Tinkerbell maps: xn+1 =x2n -yn2 +axn +byn yn+1 = 2xnyn + cxn + dyn x^m+1 = x^2m - ym2 + ax^m + by^m y^m+1 = 2x^my^m + cx^m + dy^m, in which x0, y0, x^0 and y^0 are determined and used as our generation key. We'll consider a precision of 9 decimal places for these values. The key also includes two different integers N and M which represent the number of ini- tial iterations through each map.

At this point we proceed in the following way:

  • Iterate N times the first Tinkerbell map from Eqs. (3).
  • Iterate M times the second Tinkerbell map from Eqs. (3).
  • We continue with the iterations generating two bits bi and bj given by: bi=|floor(yn 10 ^9) mod2| bj=|floor(y^n 10^9) mod2|, and ultimately outputing them. It should be mentioned that there are variations of this algorithm such as performing a XOR operation to these bits but won't be considered for this example.
  • Return to step 3 until a bit stream of the required size is generated.
Definition 4.3.1. Key space: The set of all possible keys that can be used as initial seed in a pseudo-random generator. It can be inferred from the 6 different keys used that this pseudo-random scheme will have a relatively large key space which the authors B. Stoyanov and K. Kordov estimate at 2138.
